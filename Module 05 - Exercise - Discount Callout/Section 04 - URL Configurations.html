Showing Controllers in the URL

One thing you might have noticed is that your storefront URL does not provide any hint about which controller is actually being called.

To address this and make it possible to call your own controllers through the URL, follow these steps:

  1. Go to Merchant Tools > Site Preferences > Storefront URLs.
  2. Uncheck the Enable Storefront URLs checkbox (Note: In production, this should always be checked).
  3. Click Apply.

Now, check your storefront homepage URL again. It should look like this:

Example URL: https://yourstorefront.com/controllerName/Home-Show

By following these steps, you can see which controllers are being called directly from the URL. For instance, if you're developing a custom checkout process and want to debug or verify specific controller actions, this approach can help you quickly identify the exact controller handling the request without diving into code.
